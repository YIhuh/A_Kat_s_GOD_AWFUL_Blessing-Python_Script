# A_Kat_s_GOD_AWFUL_Blessing-Python_Script
A Kat’s (GOD AWFUL!!!) Blessing original image acquisition script

Need：Python 3.13

Need to run the following dependencies:

pip install keyboard

pip install selenium

pip install requests

pip install pillow

Need to download Microsoft Edge WebDriver: https://developer.microsoft.com/en-us/microsoft-edge/tools/webdriver/ (place it in the same directory as the script)
