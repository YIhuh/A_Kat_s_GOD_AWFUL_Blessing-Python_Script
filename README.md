# A_Kat_s_GOD_AWFUL_Blessing-Python_Script
A Kat’s (GOD AWFUL!!!) Blessing original image acquisition script（From:https://a-kats-god-awful-blessing.mehgazone.com/2024/11/08/a-kats-god-awful-blessing-prologue/ ,Written by ChatGPT）

(Note: This script only fetches one image file per page,And it will skip images containing "Patreon")

Need：Python 3.13

Need to run the following dependencies:

pip install keyboard

pip install selenium

pip install requests

pip install pillow

Need to download Microsoft Edge WebDriver: https://developer.microsoft.com/en-us/microsoft-edge/tools/webdriver/ (place it in the same directory as the script)
